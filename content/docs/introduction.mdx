# Welcome to Starter Kit

Build modern web applications faster with our carefully crafted collection of reusable components, templates, and utilities designed specifically for Next.js.

## What is Starter Kit?

Starter Kit is a comprehensive collection of production-ready components that help you ship projects faster. Each component is:

- **🚀 Production Ready** - Battle-tested and optimized for real-world use
- **🎨 Beautiful by Default** - Thoughtful design with attention to detail
- **♿ Accessible** - Built with accessibility as a priority
- **🔧 Customizable** - Easy to customize and extend
- **📱 Responsive** - Works perfectly on all devices

## Quick Start

### Installation

Add components to your project using our CLI:

```bash
npx shadcn@latest add https://starter-kit-nu-two.vercel.app/r/component-name.json
```

## Example

```tsx title="image-input.tsx"
import { ImageInput } from "@/components/starter-kit-ui/image-input";

export default function ProfilePage() {
  const [avatar, setAvatar] = useState("");
  
  return (
    <div>
      <ImageInput
        value={avatar}
        onChange={setAvatar}
        name="profile-picture"
      />
    </div>
  );
}
```
